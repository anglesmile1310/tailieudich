# Tài liệu dịch
Các bài viết tổng hợp từ các trang tin mà core team Học Chủ Động cùng các thành viên thực hiện.

## Cấu trúc repos
- `docs` - Chứa các bài viết
- `images` - Chứa ảnh trong các bài viết.
